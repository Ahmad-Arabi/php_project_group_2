Database Setup Instructions
---------------------------
This folder contains the SQL file needed to set up the database for the project.

1️⃣ **Importing the Database:**
   - Open `phpMyAdmin` or any MySQL database tool.
   - Create a new database named `ecommerce`.
   - Click on the **Import** tab.
   - Select `ecommerce.sql` from this folder and upload it.
   - Click **Go** to execute the SQL script.

2️⃣ **Database Credentials:**
   - Host: `localhost`
   - Database Name: `ecommerce`
   - Username: `root`
   - Password: *(leave empty if using XAMPP/MAMP)*

3️⃣ **Important Notes:**
   - Do **not** modify `ecommerce.sql` manually unless necessary.
   - If you make schema changes, make sure to update `ecommerce.sql` for the team.
   - Contact the team before making any major modifications to the database.
